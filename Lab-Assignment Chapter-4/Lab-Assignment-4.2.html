Promises and Async/Await

What is a Promise?

A Promise represents a value that will be available in the future. It can be in one of three states:

  • Pending – operation not completed
  • Fulfilled – operation successful
  • Rejected – operation failed

Promise Example (then / catch)

let promise = new Promise((resolve, reject) => {
    resolve("Data received");
});

Async / Await Example

async/await makes asynchronous code look and behave like synchronous code, improving readability. 

async function getData() {
    let result = await promise;
    console.log(result);
}
