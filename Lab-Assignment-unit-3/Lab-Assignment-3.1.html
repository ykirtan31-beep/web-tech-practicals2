Difference Between var, let, and const

1. Scope Difference

var → Function Scope

let & const → Block Scope

function testScope() {
    if (true) {
        var a = "I am var";
        let b = "I am let";
        const c = "I am const";
    }
    console.log(a); // Works
    console.log(b); // Error
    console.log(c); // Error
}
testScope();

2. Hoisting Difference

var is hoisted and initialized as undefined

let & const are hoisted but in the Temporal Dead Zone

console.log(x); // undefined
var x = 10;

console.log(y); // Error
let y = 20;

console.log(z); // Error
const z = 30;

3. Re-declaration & Re-assignment

var p = 1;
var p = 2; // Allowed

let q = 3;
let q = 4; // Error

const r = 5;
r = 6; // Error
