JavaScript Control Flow Statements

1. if / else Statement

let age = 18;

if (age >= 18) {
    result = "Eligible to vote";
} else {
    result = "Not eligible to vote";
}

2. switch Statement

let day = 2;

switch(day) {
    case 1: text = "Monday"; break;
    case 2: text = "Tuesday"; break;
    default: text = "Invalid day";
}

3. for Loop

Best when number of iterations is known

for (let i = 1; i <= 5; i++) {
    print(i);
}

4. while Loop

Condition checked before execution

let i = 1;
while (i <= 5) {
    print(i);
    i++;
}

5. do...while Loop

Executes at least once (condition checked after)

let i = 1;
do {
    print(i);
    i++;
} while (i <= 5);
